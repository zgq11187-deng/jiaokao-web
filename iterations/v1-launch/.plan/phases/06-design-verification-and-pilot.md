# Phase 06 — Design Verification And Pilot

**Status**: `not started`
**目标**: 统一视觉与交互细节，完成验证清单，准备一个班级内测。
**前置**: Phase 01-05 完成。

## 验收判据

phase 完成 = 下面所有判据全部满足：

- `npm run check` 和 `npm run build` 通过。
- 关键页面符合 DESIGN.md：颜色、字体、圆角、边框、状态标签、微动效、无障碍目标。
- 登录、注册、章节、练习、模考、教师工作台、AIGC 生成流程均有加载、空状态、错误状态。
- 关键状态不只靠颜色表达，必须有文字。
- 受保护页面的未登录、待审核、未授权、已授权、老师角色路径均验证通过。
- AIGC 教学页生成内容有人工检查提示，不能直接视为发布。
- 准备内测说明：启动方式、访问地址、账号准备、已知限制、反馈收集方式。

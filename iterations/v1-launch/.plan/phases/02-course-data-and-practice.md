# Phase 02 — Course Data And Practice

**Status**: `not started`
**目标**: 补齐章节、题库、练习、错题、模拟考试的数据结构和 API，让学生端学习与练习有稳定数据来源。
**前置**: Phase 01 完成，受保护 API 已能区分老师、授权学生、未授权学生。

## 验收判据

phase 完成 = 下面所有判据全部满足：

- `npm run check` 通过。
- 章节列表和章节详情 API 能返回学习目标、知识点、资料数、练习数、状态等首版字段。
- 题库 API 能按章节、题型返回题目，并隐藏不应提前展示的答案解析。
- 学生提交练习后能看到结果和解析。
- 错题能按学生记录，并能在错题入口回看。
- 模拟考试能按首版组卷规则生成题目、提交、返回结果摘要和薄弱章节提示。
- 老师端可读取练习记录和错题统计所需数据。

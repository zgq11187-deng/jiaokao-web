# Phase 05 — Teacher Workbench

**Status**: `not started`
**目标**: 完成教师工作台首版：学生授权、章节内容、题库管理、布置练习、学习反馈、AIGC 生成和导出入口。
**前置**: Phase 01、02、03 完成。

## 验收判据

phase 完成 = 下面所有判据全部满足：

- `npm run check` 通过。
- 老师登录后能进入 `/teacher` 并看到学生授权、章节、题库、练习、反馈、生成、导出入口。
- 老师可审核学生申请并预录入学生账号。
- 老师可查看题库，并按章节/题型筛选。
- 老师可布置练习给授权学生或全班首版范围内的对象。
- 老师可查看学生练习记录、错题集中点和章节掌握概览。
- 老师可从章节或工作台进入 AIGC 教学页生成流程。
- 导出入口文案与 CONTENT.md 一致：Markdown / 教学网页 / PPT / 章节题库。
